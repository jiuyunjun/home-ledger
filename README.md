# 家庭记账工具

Private household accounting tool for two people.

## MVP

- Web first, Android later.
- Go backend on Cloud Run.
- Receipt image upload and OpenAI extraction.
- Optional user note during receipt upload to guide AI classification.
- Human review before ledger insertion.
- Two personal roles and one household shared role.
- Expense, income, and account-transfer records.
- JPY and CNY/RMB accounts.
- Cash, PayPay, credit card, bank, and other payment methods.
- Monthly category budgets.
- Recurring fixed expenses and recurring income.
- UTF-8 file I/O.
